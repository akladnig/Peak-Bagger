# Dashboard Wirefram
## Goal
Creates placeholder content for the Dashboard screen which has a number of placeholder cards which can be repositioned by the user using mouse drag and drop.

## Create a base card
- The base card will form the template for all other dashboard cards.
- The card will have a header row, with a CardTitle at top left.
- The header row will have a background colour of primaryContainer
- The card should be sized to allow a grid of 3x3 cards, in the full size screen.
- Add a drop shadow

## Create 6x placeholder cards with the following titles:
- Elevation
- Distance
- Latest Walk
- Peaks Bagged
- Top 5 Highest
- Top 5 Walks

## Features
- Allow each card to be repositioned by dragging and dropping.
- Other cards to be automatically repositioned.
