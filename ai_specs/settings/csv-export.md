# Peak CSV export
## Goal
Exports the ObjectBox Peak entity in CSV format

### Peak Entity Export
- Export the Peak entity with the following headers:
  - Name
  - Alt Name
  - Elevation
  - Latitude
  - Longitude
  - Area
  - Zone (gridZoneDesignator field)
  - mgrs100kId
  - Easting
  - Northing
  - Verified
  - osmId
