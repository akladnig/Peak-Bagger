# Route Editing
## Adds editing capabilty to Route Creation

- When clicking and holding on a marker change the cursor to closed hand and allow that marker to be moved.

## Delete Marker
- A single click on a marker surfaces a popup with a red Icons.deleteForever followed by red text "Delete Point"

## Undo and Redo
- Add an undo icon and redo icon to the right of the Close Loop Icon with a gap of an icon width in between.
- Undo icon is Icons.undo with toolTip "Undo (⌘ Z)"
- Eddo icon is Icons.redo with toolTip "Redo (⌘ ⇧ Z)"
- Undo undoes the last point placement, Redo adds it back.
