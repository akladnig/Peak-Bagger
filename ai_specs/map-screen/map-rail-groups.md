# Map Action Rail Grouping
## Goal
Groups FABs into logical containers.

Create the following containers containing the itemised FABs in the order shown from top to bottom:

Tools
- Import Track FAB
- Create Route FAB (does not exist - create a placeholder FAB - use same icon as Show tracks FAB)

View
- Select Basemaps FAB
- Show Map grid FAB
- Select Peaks FAB - to be renamed to "Select Peak List"
- Show tracks FAB

Loc
- Search Peaks FAB
- Goto location FAB
- Center on marker FAB
- My location FAB

Info FAB - to be placed at bottom right - no container required.
