# Updates to the Peak Lists UI
## Goal
Updates the UI by respositioning the following components:
- Peak Lists
- Import List FAB
- Mini-map
- Peak Table

## Files to examine
- 011-peak-lists-enhancements-spec.md

## Component respositioning
- Peak Lists to remain in the current position.
- Peak Lists to be displayed in a container with a border and set to an initial vertical height of 30%, not including the Peak Lists title.
- Width of Peak Lists to be set so that the width of each column adjusted so that the header content of all columns is visible with no line wrapping
- Import List FAB to display an icon only, with tooltip displaying "Import Peak List (csv)"
- Import List FAB to be repositioned to the top right of Peak Lists, inside the Peak Lists container.

- Mini-map to be positioned at bottom left inside a container with a border
- Mini-map aspect ratio to be set to 4:3
- Mini-map container to initially occupy 70% of the vertical height.

- Peak Table to be positioned on the right hand side, and width of each column adjusted so that the header and content of all columns is visible with no line wrapping.
- Peak Table to be wrapped in a container.
