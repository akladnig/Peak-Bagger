# Goal
Add a new peak list by selecting peaks as per the "Add new peak" FAB

## Requirements
- Add a new button to the left of "import Peak List" to "Add New Peak List", using the same icon as "Add New Peak"
- Show a dialog box with a title "Peak List Name" and allow text entry of a List Name. Use same styling and validation as per the "Import Peak List" List name entry.
- Dialog box should have a cancel button and a Create button
- On clicking create bring up the peak selector as per "Add new peak"
