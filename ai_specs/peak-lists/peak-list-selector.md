# Peak List Selector
## Goal
Changes the functionality of the Show Peaks FAB to allow the user to select which Peak List to display on the map screen.

- The Show Peaks FAB is to be renamed to Select Peaks including the tooltip.

- When the Select Peaks FAB is pressed all Peak Lists in teh ObjectBox PeakList entity plus "None" and "All Peaks" are to be displayed using the same pattern as the Select BaseMaps FAB.

1. None turns off display of peaks.
2. All Peaks displays every peak in the ObjectBox entity Peak.

- Items 1. & 2. above provide the exact functionality as the current Show Peaks functionality. 

- The Peak Lists are to be sorted alphbetically except for None which is at the top of the list and All Peaks at the bottom.
