# Peak List Selection
## Goal
When adding a New Peak to a list allow multiple peaks to be selected from the list.

### Requirements
- A checkbox should be shown to left of the peak name to show the peaks that have been selected.
- Peaks should be sorted alphabetically
- Display of Name, height and map to be collapsed to a single row
- if a selected peak(s) is already in the selected peak list, skip it
- increase the height of the list so that it fills up to the Add New Peak FAB
