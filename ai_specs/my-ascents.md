# My Ascents Card
## Goal
Creates a My Ascents Card that displays a list of Ascents.
This is similar card to My Lists card so should be able to reuse a good % of that card including providers for the data sources.

Displays a table with following columns:
- Peak Name
- Elevation
- Date Climbed

The rows are to be sorted by Date Climbed descending. Add a sort button to allow ascending/descending sort by Date Climbed.

Display a Year header row and group ascents for that year under the header row.

Reuse the Top 5 Walks Card and rename to My Ascents.

Migrate the dashboard slot id to "my-ascents"

The objectBox PeaksBagged entity is the source of truth.
