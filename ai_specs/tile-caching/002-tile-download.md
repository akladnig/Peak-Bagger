# Goal
Use local tiles for rendering the map as a first priority then download from the URL as required.
- When the selected basemap is "Tracestrack Topo" check the assets/map_tiles/OSM_tracestrack folder for a tile matching the current zoom level and location.
- When the selected basemap is "OpenStreetMap" check the assets/map_tiles/OSM_standard folder for a tile matching the current zoom level and location.
- When the selected basemap is "Tasmap Topgraphic" check the assets/map_tiles/tasmap_topo folder for a tile matching the current zoom level and location.
- When the selected basemap is "Tasmap 50k" check the assets/map_tiles/tasmap_50k folder for a tile matching the current zoom level and location.
- When the selected basemap is "Tasmap 25k" check the assets/map_tiles/tasmap_25k folder for a tile matching the current zoom level and location.
- A check for tiles is required whenever a pan or zoom is carried out.
- A check for tiles is required whenever a new basemap is selected.
