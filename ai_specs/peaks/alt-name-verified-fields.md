# Goal
Adds two new fields to the objectBox entity Peak

1. Add a new field to the objectBox Peak entity named "verified" which is a boolean and defaults to false.
- This should be displayed as a checkbox in ObjectBox Admin in both the table and details sidebar.
- it is manually set by the user to identify if peak name, height and location have been verified.
- it should appear at the bottom of the details under northing

2. Add a  new field to the objectBox Peak entity named "altName" with a title of "Alt Name" which is an empty string by default. It should appear under the Name field in the details sidebar and to the right of the name column in the table view.
