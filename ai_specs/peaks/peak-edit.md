# Peak Edit
Allows Peak Metadata to be directly edited from the peak info popup on Map Screen

Changes the behaviour of the edit button to allow specific local edits.
When the edit button is clicked:
- Allow editing the Peak Name and Height via Text Fields
- Add a new line under the title row and above Height, showing 3x icons:
	- A Peak Icon a right arrow Icon and a Marker Icon with tooltip text "Move Peak to Marker"
- On clicking in the region of the 3x Icons update the lat/long and MGRS coordinates of the peak.
- Add new button at the bottom of the popup "Edit in Peak Admin". This will navigate to ObjectBox Admin, open the Peak entity entity, select the peak to be edited, enter edit mode directly and update the Search field with the peak name.
