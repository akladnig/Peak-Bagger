# Peak Info updates
## Goal
- Adds a Drop Marker button to the peak info popup
- Adds My Ascents
- Fix display format

## Drop Marker
- Add a new button titled "Drop Marker" with tooltip "Drop a Marker on the Peak"
- Button icon is the same as the exsiting marker including colour
- To be placed to the left of the close icon on the same row
- Add a tooltip to the close icon "Close Peak Info"

## My Ascents
- If the peak has a recorded Ascent then display:
  - "My Ascents: "
  - then a list of trackName and trackDate
- the first Ascent is to be placed under "My Ascents" and indented
- to be placed under "Height:"

## Display Format
- use the formatElevationMetres for the Height display
- All values associated with labels such as Height:, Map: etc. to be in Bold text. No change to the label text style.
