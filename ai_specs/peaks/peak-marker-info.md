# Peak Marker Update
Adds Peak Name and Height to the Peak Markers

- Add a setting to the Settings Screen as a simple switch - "Show Peak Info"
- If the switch is set show the name of the peak and height as centered column in the following order:
  - marker
  - name
  - height
- Create an outlined text style for the name and height and save it to theme.dart. The text should be set to surface and the outline to onSurface. Text size is bodySmall.
