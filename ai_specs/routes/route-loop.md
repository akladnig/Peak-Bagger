# Route Close Loop
## Goal
Adds a new button "Close Loop" using icons.refresh

When clicking on this button a route back to the starting point from the current point is created.
If there is no loop available (i.e. alternate route back to start) fallback to "Out and Back".
If off track fallback to straight line mode.

The Icon is to be placed in between the "Out and Back" and the Route Name text Box.
