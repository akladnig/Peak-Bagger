# Elevation Profile
## Goal
Creates an Elevation Profile which will be viewable in the following:
- Track and Route info pop ups
- Route Creation Bottom sheet
- New Latest Walk details popup - future, not yet implemented.

## Requirements
- To be displayed directly under the Elevation header in the Track and Info popup
- To be displayed in the bottom section of the Create Route bottom sheet
- Uses the same graph style as the graphs in the Dashboard screen - Elevation, Peaks Bagged etc.
- Y-axis is as per the Dashboard Elevation graph.
- X-axis is either time or distance, default is distance.
- This is simple plot of distance/elevation or time/elevation
