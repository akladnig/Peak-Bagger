# Route Out and Back
## Goal
Adds a new button "Out and Back" using icons.syncalt with tooltip "Out and Back".

When clicking on this button a waypoint is created at the current location and then the route continues following the original route in reverse back to the starting point, so that the route is symmetrical around the waypoint.

If the waypoint is a peak set the <name> tag to the peak name, otherwise set the name to "Waypoint x", where x starts at 1 and increments fot additional waypoints added manually (currently out of scope)
The Icon is to be placed in between the Straight Line button and the Route Name text Box.
